NEXT STEPS:

1. Extract this archive:
   unzip RuntipiOS-Complete-Ready.zip

2. Initialize git:
   cd runtipi-os
   git init
   git add .
   git commit -m "Initial RuntipiOS"

3. Add Buildroot submodule:
   git submodule add https://github.com/buildroot/buildroot.git buildroot
   git submodule update --init

4. Make scripts executable:
   chmod +x board/runtipi/rootfs-overlay/usr/local/bin/*.sh
   chmod +x board/runtipi/post-build.sh

5. Add your Runtipi logo:
   cp /path/to/logo.png board/runtipi/rootfs-overlay/usr/local/share/runtipi/captive-portal/logo.png

6. Build for Raspberry Pi:
   cd buildroot
   make BR2_EXTERNAL=.. -C . rpi4-64_defconfig
   make -j$(nproc)

7. Flash to SD card:
   dd if=buildroot/output/images/runtipi-rpi4-64.img of=/dev/sdX bs=4M status=progress
   sync
