# RuntipiOS - Buildroot-Based Embedded OS for Runtipi

## Project Structure

This archive contains all necessary files to build RuntipiOS. 

### Contents:

- **Configuration**: Buildroot defconfig and system configuration
- **Scripts**: Systemd services and shell scripts for boot process
- **Captive Portal**: Web interface for Wi-Fi setup (multilingue)
- **Documentation**: README and project guidelines
- **CI/CD**: GitHub Actions workflow for multi-architecture builds

### Quick Start:

1. Extract this archive
2. Navigate to the project directory
3. Run: ./scripts/setup-project.sh
4. Follow the instructions in README.md

### File Structure:

```
runtipi-os/
├── board/runtipi/
│   ├── configs/                 # Buildroot configurations
│   ├── rootfs-overlay/          # System files
│   │   ├── etc/                 # System configuration
│   │   ├── usr/local/bin/       # Scripts
│   │   └── usr/local/share/     # Assets (logo, portal)
│   ├── post-build.sh            # Post-compilation script
│   └── linux-defconfig          # Kernel config
├── .github/workflows/           # GitHub Actions
├── scripts/                     # Helper scripts
├── README.md                    # Full documentation
└── buildroot/                   # Buildroot submodule (clone separately)
```

### Important Notes:

- Add your Runtipi logo to: board/runtipi/rootfs-overlay/usr/local/share/runtipi/captive-portal/logo.png
- Clone Buildroot as a git submodule
- All shell scripts must be made executable after extraction

### Supported Features:

✅ Buildroot-based lightweight Linux
✅ Automatic Runtipi installation
✅ WiFi captive portal (multilingue)
✅ Web interface (no SSH required)
✅ Fast boot (15-20 seconds)
✅ Multi-architecture support

### Next Steps:

1. Extract the archive
2. cd runtipi-os
3. git init
4. git submodule add https://github.com/buildroot/buildroot.git buildroot
5. cd buildroot
6. make BR2_EXTERNAL=.. -C . rpi4-64_defconfig
7. make -j$(nproc)
